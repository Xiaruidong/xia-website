# 微信公众号文章爬虫

用于从公众号文章中提取投递链接的工具

## 方法一：自动爬取（推荐尝试）

### 安装依赖

```bash
cd wechat_crawler
pip install -r requirements.txt
```

### 使用方法

```bash
python wechat_article_crawler.py
```

### 功能

- 通过搜狗微信搜索获取公众号文章
- 自动识别并提取招聘投递链接
- 支持的平台：牛客网、猎聘、BOSS直聘、拉勾、前程无忧等
- 结果导出为CSV文件

### 注意事项

⚠️ **搜狗微信搜索可能有反爬限制**：
- 如果请求过快会被临时封禁
- 建议增加请求间隔时间
- 可能需要手动验证

## 方法二：手动导出（更稳定）

### 步骤1: 获取文章链接

**方式A：使用新榜等第三方平台**
1. 访问 https://www.newrank.cn
2. 搜索公众号名称
3. 查看历史文章列表
4. 复制文章链接

**方式B：微信内置浏览器**
1. 在微信中打开公众号
2. 进入"全部消息"
3. 在浏览器中打开（复制链接到外部浏览器）

### 步骤2: 使用提取工具

将文章链接保存到 `article_links.txt`，每行一个：

```text
https://mp.weixin.qq.com/s/xxxxx
https://mp.weixin.qq.com/s/yyyyy
...
```

然后运行提取脚本：

```bash
python extract_links.py
```

## 常见问题

### Q: 搜狗搜索找不到文章？
A: 尝试以下方法：
- 使用更精确的搜索关键词
- 直接在搜狗微信搜索页面手动搜索
- 使用第三方平台（新榜、西瓜数据）

### Q: 提取不到投递链接？
A: 可能原因：
- 文章中的链接被平台过滤
- 链接格式不常见
- 需要调整正则表达式模式

### Q: 如何获取公众号完整文章列表？
A: 完整列表需要：
- 公众号运营者权限
- 或使用第三方数据服务（付费）
