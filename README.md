# 義 - 个人网站

一个文艺冬日森系风格的个人网站，静谧林间雪景背景，落雪、枯枝、薄雾氛围感。

## 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite
- **路由**: Vue Router
- **数据存储**: localStorage

## 功能特性

- 🏠 **首页**: Hero 区域、介绍卡片、最新文章、生活碎片
- 📝 **博客**: 文章列表、分类筛选、文章详情
- 🖼️ **画廊**: 图片展示、Lightbox 效果
- 👤 **关于**: 个人介绍、时间线、技能兴趣
- 🔐 **后台管理**: 文章管理、画廊管理

## 开始使用

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 后台管理

- **访问地址**: `/admin`
- **默认密码**: `admin123`

## 设计风格

- **主色调**: 米白 + 浅青灰 + 霜蓝色
- **风格**: 冬日森系、静谧、极简
- **特色**: 飘雪动画、柔和光影、毛玻璃效果
