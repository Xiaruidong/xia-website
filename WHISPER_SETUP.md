# OpenAI Whisper 录音转文字工具

使用 OpenAI Whisper API 将语音录音转换为文字。

## 功能特点

- 🎤 **实时录音** - 直接在浏览器中录制音频
- 📁 **文件上传** - 支持上传预录制的音频文件
- 🌍 **多语言支持** - 支持中文、英语、日语等 15+ 种语言
- 📝 **多种输出格式** - 纯文本、JSON、SRT 字幕
- 📋 **一键复制** - 快速复制转录结果
- 💾 **保存到画廊** - 将转录内容保存到本地画廊
- 📥 **导出功能** - 导出为 TXT、JSON、SRT 文件
- 📜 **历史记录** - 自动保存最近的 20 条转录记录

## 使用方法

### 1. 获取 API Key

1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册/登录账号
3. 在 API Keys 页面创建新的 API Key
4. 复制 API Key（格式：`sk-...`）

### 2. 开始使用

#### 方式一：实时录音

1. 在工具页面输入 OpenAI API Key
2. 点击 "开始录音" 按钮
3. 对着麦克风说话
4. 点击 "停止录音" 结束录音
5. 点击 "开始转录" 等待结果

#### 方式二：上传音频文件

1. 在工具页面输入 OpenAI API Key
2. 点击或拖拽上传音频文件
3. 点击 "开始转录" 等待结果

### 3. 配置选项

- **语言**：可选择特定语言或留空自动检测
- **提示词**：可提供上下文提示提高准确率（可选）
- **响应格式**：选择输出格式
- **温度**：控制输出随机性（0-1，默认 0）

## 支持的音频格式

- MP3, MP4, MPEG, MPGA, M4A, WAV, WEBM
- 文件大小限制：25MB

## 支持的语言

中文、英语、粤语、日语、韩语、西班牙语、法语、德语、意大利语、葡萄牙语、俄语、阿拉伯语、印地语、泰语、越南语等

## 费用说明

Whisper API 按音频时长计费：
- $0.006 / 分钟 (默认模型 whisper-1)
- 例如：10 分钟录音约 $0.06

详细价格请查看 [OpenAI Pricing](https://openai.com/pricing)

## 注意事项

- API Key 会保存在本地浏览器中，请勿在公共设备上使用
- 转录时间取决于音频长度，请耐心等待
- 建议使用 Chrome、Edge 等现代浏览器以获得最佳录音体验
- 需要授予麦克风权限才能使用录音功能

## 技术实现

- **前端录音**：Web Audio API + MediaRecorder
- **API 调用**：OpenAI Whisper API v1
- **音频处理**：FormData 上传，支持多种编码格式
- **存储**：LocalStorage + Supabase 画廊

## 常见问题

### Q: 录音没有声音？
A: 请检查浏览器麦克风权限设置，确保已授权访问麦克风。

### Q: 转录失败？
A: 请检查：
1. API Key 是否正确
2. 账户是否有足够余额
3. 网络连接是否正常
4. 音频文件格式是否支持

### Q: 如何提高转录准确率？
A: 
1. 选择正确的语言
2. 使用高质量的录音设备
3. 减少背景噪音
4. 添加提示词（prompt）提供上下文

### Q: 支持实时转录吗？
A: 当前版本不支持实时转录，需要先录音/上传完整音频文件。
